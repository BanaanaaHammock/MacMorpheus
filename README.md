## Supported modes
* 3D 360° Horizontal (Stacked)
* 3D 180° Vertical (Side By Side)

## Controls
* **Enter** - Toggle fullscreen
* **Space** - Play / Pause
* **Arrows** - Skip backward / forward
* **R** - Center view / reset orientation
* **ESC** - Exit fullscreen / quit
* ***Mouse drag*** - Look around
